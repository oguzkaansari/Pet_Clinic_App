# Pet Clinic Web Applciation with Spring MVC

This web applciation is designed as a veterinary clinic automation system that includes admin, doctor and secretary tools.

## Languages Used

| Language | % |
| ------------- |:-------------:|
| Java | 50 |
| SQL (MySql) | 5 |
| Javascript | 30 |
| HTML/CSS | 15 |
